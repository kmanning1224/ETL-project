June Unemployment:

Data for CA, FL & NY


The purpose of this project was to review the unemployment rates for counties from 3 States:

  • California
  • Florida
  • New York

so further analyses could be done on potential impact from COVID-19 and the associated effects of unemployment itself.


Data for this project was acquired from 3 sources:

  1. CA Source
  2. FL Source
  3. NY Source

Data was cleaned by removing all NaN values. Columns were renamed and indexes were reset. Dataframe was concatenated so that further study could be ran on each state's data AND the concatenated dataset as well.


Project by Katherine Manning & Sofia Sotillo.
